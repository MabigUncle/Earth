插件名称：基础标绘插件 CommonPlot
插件版本：V4.0，同样适配数字地球
基础环境版本：Vue 2.6.11、Vue-cli 3.X、Cesium 1.73
公司：GEOVIS 中科星图
作者：王照方
功能说明：
    1、支持点标和三维模型的绘制和二次编辑，点图标和三维模型的上传、预览、删除、重命名、组织管理（需要后台jar包服务、mysql数据库、nginx支持）。
    2、支持折线的绘制和二次编辑。
    3、支持多边形的绘制和二次编辑。
    4、支持矩形的绘制和二次编辑。
    5、支持圆形的绘制和二次编辑。
    6、支持文字的绘制和二次编辑。
    7、支持态势（直线箭头、攻击箭头、钳击箭头）的绘制和二次编辑。
    8、支持标绘的清单列表：命名、定位、显隐、删除。
    9、支持标绘内容的导入和导出：本地的json文件。
    10、支持快捷键操作：CTRL+C复制、CTRL+V粘贴、DELETE删除、鼠标滚轮缩放（点标）。
    11、标绘方式有3种：鼠标地图交互绘制、十进制经纬度输入、度分秒经纬度输入。

操作说明：
    1、单击面板的“开始绘制”按钮，开始绘制图形。鼠标左键单击绘制折点，鼠标右键单击结束绘制。
    2、双击选中三维球上的某个标绘对象，可以进入编辑模式（锁定镜头），并打开属性编辑面板。
    3、双击选中三维球上的某个标绘对象，按CTRL+C键。然后在需要复制的地方鼠标单击后按CTRL+V键，即可实现复制、粘贴。
    4、双击选中三维球上的某个标绘对象，按DELETE键即可删除此标绘。
    5、关闭编辑面板或双击三维球的空白处，即可解除镜头锁定。

API参数说明：
    1、PointImageSelect组件的imageUrl：获取图标的cms图标后台管理地址 http://localhost:9998

API方法说明：
    1、获取标绘场景序列化Json字符串：this.$refs.CommonPlot.$refs.PlotListTab.plotTransToJson()

Vue事件说明：
    1、CloseEditTab：关闭编辑面板事件
    2、clearAllPlot：清除所有标绘实体事件
    3、deleteNodeEntity：在树节点上点击删除按钮，删除某个实体事件
    4、OpenEditTab：打开编辑面板事件
    5、deleteKeyEntity：键盘按下delete键事件
    6、addEntityPlot：增加一个实体事件
    7、ChangeSelectedImgScale：改变点标的大小事件
    8、ChangeSelectedModelScale：改变模型的大小事件（gltf）
    9、CloseCommonPlotTab：关闭标绘面板事件
    10、OpenPointImageTabShow：打开图标选择面板事件

Version变更说明：
    1、增加了模型的管理和标绘功能
    2、增加了标绘清单面板组件的ref属性
    3、增加了标绘面板关闭事件
    4、修复了一些bug

注意事项：
    1、此版本暂不支持在地形上的标绘。

图标管理系统与标绘系统的关系说明：
	图标管理系统为标绘系统提供丰富的、用户自定义的图标（图片和三维模型）。
